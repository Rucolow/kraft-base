# KRAFT BASE 周辺案内・防災情報（マニュアル §03 / §12 追補）＋ 火打石の一般作法

所在地：和歌山県新宮市熊野川町上長井字小和瀬134番地3（小口・小和瀬地区）。

## §03 追補：周辺案内

### 熊野古道
- **小雲取越（小口 → 請川／本宮方面）**：約13km、標準歩行 約4時間30分（休憩別、実所要5〜6時間）。最低標高約60m（小和瀬）〜最高約500m（百間ぐら）。小和瀬から桜茶屋への登りが一番きつく、以降は緩やか。ハイライトは展望地「百間ぐら」。
- **大雲取越（那智大社 → 小口）**：約14.5km、歩行7〜9時間（標準歩行約5時間）。最高約870m（越前峠付近）。**中辺路最難関**。円座石・舟見茶屋跡などを通る。
- **共通**：古道上に飲食店・売店・自販機はない。昼食・水は出発前に用意してもらう。
- **古道入口**：小口自然の家から約1kmの舗装路、小口川の橋のたもとが取付き（小和瀬）。KRAFT BASE から近い。橋のたもとに旧渡し船跡。

### 交通（バス）
- JR新宮駅 →（熊野御坊南海バス／奈良交通バス）「**神丸**」約40分 → 「**小口**」行きに乗り換え約15分。
- 本宮方面 → 「神丸」約45分 → 「小口」行き約15分。
- 小口↔新宮、小口↔本宮はいずれも **神丸（または志古）で乗り換え**。便数は少ない。
- **【要確認】最新の時刻表**（季節・改正で変動）。受付に最新の紙時刻表を常備し、前夜のお見送り時に翌朝の便を伝える。

### タクシー

### 買い物・売店
- **小口にコンビニ・スーパー・商店はない。** 最寄りは新宮方面で車・バスで約1時間。直近店舗と距離は **【要確認】**。

## §05 / §08 追補：火打石（切り火）の一般的な作法

KRAFT BASE 固有の石・道具・保管場所・声かけスクリプト・辞退時の対応は、モーリーと実演して確定する（既存の確認事項）。アプリには下記の一般作法を載せておく。

切り火（きりび）は、出かける人の安全を願い、その背後で火打石と火打金を打ち合わせて火花を散らす日本古来の習俗。

一般的な進め方：
1. 出かける人に背を向けて立ってもらう（または横に立つ）。
2. 火打石（燧石）に火打金（鋼）を、相手の右肩越し・背中の方へ向けて「カチッ」と数回（目安2〜3回）打ち下ろし、火花を散らす。火花は相手に当てず、軽く後方へ。
3. 「お気をつけて」「良いご旅を」と声をかける。
4. 火花が穢れ・災いを払うとされる。厳密な回数・所作は地域や流儀で異なり、固定の決まりはない。

注意：火気のため、可燃物のない場所で行う。火花のみで炎は出ない。
